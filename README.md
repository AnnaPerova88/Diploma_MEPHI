# Diploma_MEPHI

📌 Diploma_MEPHI

**Коммуникативная AI-система с цифровыми аватарами для подготовки к интервью**

**📖 Описание проекта**

В рамках дипломной работы разработана коммуникативная система на основе искусственного интеллекта, объединяющая:

* большие языковые модели (LLM)
* цифровых аватаров (видео)
* структурированные сценарии интервью

**Система предназначена для подготовки к собеседованиям в области Data Science и моделирует взаимодействие пользователя с HR-специалистом или AI-лектором.**

**🎯 Идея проекта**

Основная идея — создать систему, которая сочетает:

* 🤖 диалог с LLM (HR / лектор)
* 🎥 видео-аватары (HeyGen / D-ID)
* 📚 базу вопросов для интервью
* 💬 автоматическую генерацию обратной связи

**Архитектура системы**

* Пользователь
* ↓
* Web-интерфейс (Gradio)
* ↓
* Сценарный модуль (вопросы интервью)
* ↓
* LLM (GigaChat API)
* ↓
* Модуль аватаров (HeyGen / D-ID)
* ↓
* Генерация обратной связи
